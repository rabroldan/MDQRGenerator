About Us

This software is created to transform the way notes, prescriptions, or important documents are handled. It converts handwritten notes, and electronic prescriptions into QR codes. This app offers a safe way to exchange and store private data, but it also guarantees the accuracy and legitimacy of important information including the date of issue, patient information, physician information, medication information, and more.

Local Development

Unzip the contents of zip file in a local folder.

Install the following Python libraries (see requirements.txt for respective versions):

  • pip install flask
  • pip install pytesseract
  • pip install fpdf2
  • pip install PyPDF2
  • pip install pillow
  • pip install qrcode

Download and install Tesseract.

Locate extractor.py and change “pytesseract.pytesseract.tesseract_cmd” (line 11) to the appropriate path of tesseract.

Execute python QRgen.py on the command line.

Using the Application

Uploading Documents: Upload images or scanned documents for text extraction via the upload_image() function.

Data Extraction: The extractor.py module processes uploaded images to extract text, which can be downloaded as plain text or PDF using download_text() and download_text_pdf() functions, respectively.

QR Code Generation: The application generates a QR code containing the extracted data. This QR code can be downloaded as an image (qr_code.png) or as a PDF (qr_code.pdf) using the download_pdf() function.

Security and Verification:

The application ensures the security and verification of the information by encrypting the extracted data and embedding it within the QR code.

Outputs and Artifacts

The application produces several outputs, including:

  • Text File: A plain text file containing the extracted information (extracted_text.txt).
  • QR Code Image: A PNG file of the generated QR code (qr_code.png).
  • QR Code PDF: A PDF file with the QR code, suitable for printing or sharing (qr_code.pdf).
